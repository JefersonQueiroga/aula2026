Tipos de Kernel

Monolítico vs Micro vs Híbrido

🐧

Monolítico

Tudo dentro do núcleo

⚡ Alto desempenho
⚠️ Mais complexo

Ex: Linux

🔬

Microkernel

Só o essencial no núcleo

🛡️ Mais seguro
🐢 Menos veloz

Ex: MINIX

🪟

Híbrido

Mistura dos dois modelos

⚖️ Equilíbrio

Ex: Windows

Conceitos Fundamentais

🔄 Concorrência

  • Vários processos disputam recursos
  • Alternância rápida na CPU
  • Possível com 1 núcleo
  • Ilusão de simultaneidade
VS

⚡ Paralelismo

  • Execução simultânea real
  • Múltiplos núcleos de CPU
  • Requer hardware multi-core
  • Simultaneidade real

Memória Virtual

Paginação

A memória é dividida em blocos chamados páginas

RAM Cheia?

🧠💨

Páginas menos usadas...

→

Vai para o Disco

💾

...são movidas para o disco

Esse processo é chamado de paginação (swap)

Problema

Fragmentação de Memória

Quando a memória é utilizada de forma desorganizada

Interna

📦🔲

Espaço desperdiçado dentro dos blocos alocados

Externa

🧩🧩

Espaços livres espalhados entre blocos

Impacto: Redução de eficiência do sistema
